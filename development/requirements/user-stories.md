# User Stories

## Unauthenticated User

1. As an unauthenticated user, I want to be able to view the list of available products, so that I can browse through them before deciding to create an account.

2. As an unauthenticated user, I want to be able to view the details of a specific product, so that I can get more information about it.

3. As an unauthenticated user, I want to be able to register for an account, so that I can place orders and access additional features.

4. As an unauthenticated user, I want to be able to log in to my existing account, so that I can access my profile and previous orders.

## Member (Authenticated User)

1. As a member, I want to be able to update my profile information, so that I can keep it accurate and up-to-date.

2. As a member, I want to be able to view my order history, so that I can track my previous orders.

3. As a member, I want to be able to place a new order, so that I can request a product from a baker.

4. As a member, I want to be able to edit or cancel an existing order, so that I can make changes if necessary.

> To be added: cancel an order, edit an order, rate and review order, ...

## Baker (Authenticated User)

1. As a baker, I want to be able to create and manage my own profile, so that I can showcase my products and services.

2. As a baker, I want to be able to add new products to my catalog, so that users can place orders for them.

3. As a baker, I want to be able to view and manage incoming orders, so that I can fulfill them in a timely manner.

4. As a baker, I want to be able to update the status of an order (e.g., accept, reject, start preparation, order ready), so that the member can be informed of the progress.

5. As a baker, I want to be able to receive notifications for new orders and order updates, so that I can stay informed even when not actively using the platform.
